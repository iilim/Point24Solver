# ==========================================
# Point24Solver 依赖清单
# ==========================================
# 本项目无需安装任何第三方包，全部使用 Python 标准库实现。
# 请确保您的 Python 版本 >= 3.8 (因为使用了新的 ast.Constant 特性)。
#
# 核心依赖的标准库模块说明：
# ast      - 用于构建抽象语法树，实现安全的表达式解析与防注入校验
# json     - 用于历史最高分等游戏状态的本地持久化存储
# os       - 用于文件路径处理与状态文件读取
# random   - 用于互动模式中随机生成有效的卡牌组合
# re       - 用于 CLI 模式中正则解析用户输入的 /set 配置指令
# tkinter  - 用于构建现代扁平化风格的 GUI 图形界面 (通常Python自带)
# ==========================================
